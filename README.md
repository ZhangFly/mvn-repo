# mvn-repo
添加 ZFly 的 maven 仓库

```
<repositories>
    <repository>
    	<id>ZFly</id>
        <url>https://raw.githubusercontent.com/ZhangFly/mvn-repo/master</url>
    </repository>
</repositories>
```

### wtfsocket-server

```
<dependencies>
	<dependency>
    	<groupId>ZFly</groupId>
        <artifactId>wtfsocket-server</artifactId>
        <version>1.0.0</version>
    </dependency>
</dependencies>
```
